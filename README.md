# Настройка проекта

```shell
    pip install -r requirements.txt
```

Создать файл ``.env`` в корневой директории проекта или передать переменные среды при запуске контейнера и определить там переменные:
* ``URL`` - адрес адаптера
* ``TOKEN`` - токен бота, выданный вк
* ``BACKEND_URL`` - адрес главного сервера

При сборке Docker-контейнера использует порт 3000
